# BuilderAI - AI Web App Builder

[![BuilderAI Demo](https://img.shields.io/badge/Live%20Demo-Visit%20Site-blue?style=for-the-badge)](https://019f155b-8ef0-7c0d-99d8-f5847761aff0.arena.site)

BuilderAI is an AI-powered web application builder that enables users to create full-stack web applications through natural language prompts. Built on the Babel Town platform, it transforms descriptions into working code with a clean, modern interface.

## ✨ Features

- **Natural Language to Code**: Describe what you want to build, and BuilderAI generates the complete application
- **Live Preview**: Real-time preview of your generated app with desktop, tablet, and mobile viewports
- **Multi-View Interface**: 
  - Chat view for describing your app
  - Code view to see/edit generated code
  - Preview view to test your application
- **Pre-built Templates**: Quick-start templates for common applications:
  - Todo apps with clean modern UI
  - Weather dashboards with charts
  - SaaS landing pages
  - Calculators with dark theme
  - Pomodoro timers
  - Recipe card galleries

## 🚀 Quick Start

1. Visit the [live demo](https://019f155b-8ef0-7c0d-99d8-f5847761aff0.arena.site)
2. Enter your API key (Babel Town integration)
3. Describe your desired app in plain English
4. Watch as BuilderAI generates your application
5. Preview, edit, and deploy your creation

## 🛠️ Tech Stack
